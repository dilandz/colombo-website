# Discover Colombo: City Travel Guide

## Overview
Discover Colombo is a responsive, multi-page travel guide designed to provide visitors with essential insights into Colombo, Sri Lanka. The project features neighborhood statistics, cultural highlights, and interactive UI elements, built to demonstrate full-stack web development fundamentals and responsive design principles.

## 🚀 Live Demo
[View the live project here](https://dilandz.github.io/colombo-website/)

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Architecture:** Component-based UI design
* **DevOps (In Progress):** Docker containerization & CI/CD pipeline integration

## Key Features
* **Multi-page navigation:** Dynamic routing for seamless exploration.
* **Responsive Layout:** Optimized for all screen sizes (mobile, tablet, desktop).
* **Interactive UI:** Engages users with data-driven insights about Colombo's diverse neighborhoods.

## 🛠️ DevOps & Infrastructure Roadmap
I am currently evolving this project from a static site to a fully containerized application. The following features are in development:
* [ ] **Dockerization:** Packaging the application for consistent cross-environment deployment.
* [ ] **CI/CD Integration:** Implementing GitHub Actions to automate build and deployment processes.
* [ ] **Cloud Hosting:** Migrating to a scalable cloud infrastructure (AWS/GCP).

## What I Learned
* **System Thinking:** Moving beyond static web pages to consider infrastructure, containerization, and the full lifecycle of an application.
* **Component Architecture:** Refactoring code for better maintainability and scalability.
